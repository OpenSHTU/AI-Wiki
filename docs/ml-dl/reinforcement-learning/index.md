# Reinforcement Learning

本页是 `Reinforcement Learning` 的目录索引。

## 简述

- 通过 agent 与 environment 的交互学习策略。
- 策略形式：$\pi(a|s)$
- 优化目标：$\max_\pi \mathbb{E}\left[\sum_{t=0}^{T}\gamma^t r_t\right]$
