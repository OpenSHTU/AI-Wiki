# 偏导数

## 它在 AI 中解决什么问题？

占位说明：解释多参数模型中如何考察 loss 对某一个参数或某一组参数的局部变化。

## 核心概念

- 待补：partial derivative
- 待补：multivariable function
- 待补：coordinate-wise change

## 关键公式

待补：$\frac{\partial \mathcal{L}}{\partial \theta_i}$。

## 它出现在模型的哪里？

- 待补：neural network parameter gradient
- 待补：linear layer backward
- 待补：embedding table update
